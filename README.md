# SRK Movies Collection Dashboard 🎬

## Project Overview

This project analyzes the movie collection of Shah Rukh Khan (SRK) using Power BI. The dashboard provides insights into movie performance, success ratio, yearly trends, and box office collections.

The goal of this project is to visualize movie data and understand hit movies, flop movies, total collections, and performance trends over the years.

## Dashboard Features

## Key Performance Indicators (KPIs)

- Total Collections: ₹3,892.85 Crores
- Total Movies: 63
- Hit Movies: 33
- Flop Movies: 30
- Success Ratio: 52.38%
- Active Years: 31

##Visualizations Included

1. Top 5 Movies

Shows the highest box office collection movies.

Example: Jawan, Pathaan, Chennai Express.

2. Movie Count by Verdict

Displays the distribution of movies based on verdict categories:

- Blockbuster
- Super-Hit
- Hit
- Average
- Flop

3. Bottom 5 Movies

Shows the lowest box office collection movies.

4. Movie Count by Year

Line chart displaying the number of movies released per year.

5. IMDb Rating Table

Displays movie names with their corresponding IMDb ratings.

## Tools and Technologies Used

- Power BI – Dashboard creation and visualization

- DAX – Calculated measures and KPIs

- Excel / CSV Dataset – Data source

## Key Insights

-SRK has acted in 63 movies over 31 active years.

-The success ratio is 52.38%, showing more than half of the movies were successful.

-Movies like Jawan and Pathaan generated the highest box office collections.

-Some movies performed poorly at the box office but still contributed to SRK's long career span.

## Dataset Source

The dataset used in this project was compiled from publicly available movie data websites such as IMDb and Wikipedia. The data was cleaned and organized into a structured format for analysis and visualization in Power BI.
The dataset contains the following information:
- Movie Name
- Release Year
- Box Office Collection
- IMDb Rating
- Movie Verdict (Hit / Flop / Blockbuster)
